java DataSource 配置

数据库环境配置的含义

  • 例如：

1: oracle.jdbc.OracleDriver 是连接数据库的驱动
2: jdbc:oracle:thin:@192.168.88.15:1521:XE 数据库连接url包含端口（1521），连接数据库名（XE）
3: username：数据库连接名字
4：password: 数据库连接密码
